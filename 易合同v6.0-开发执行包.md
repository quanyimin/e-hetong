## 四、插件配置模板

详见 `lib/scenes/` 目录下的场景配置。

### 行业插件注册格式

```typescript
// lib/scenes/types.ts
export interface ScenePlugin {
  code: string           // 行业代码：LANDLORD, RESTAURANT, 
  name: string           // 中文名
  icon: string           // Lucide 图标名
  description: string    // 描述
  routes: {              // 前端路由
    path: string
    name: string
    icon: string
  }[]
  configSchema: Record<string, any>  // 动态字段定义
}
```
