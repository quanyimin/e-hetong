## 一、本次调研的核心发现

经过对全球前沿技术五个维度的深度调研，以下是与你需求最相关的革命性发现：

### 发现1：GenUI（生成式UI）— 2026年最大技术变量

2026年，Google Research 已在 Gemini 中落地 **「完全个性化生成界面」** ，不是简单的A/B测试，而是 **每次对话都生成不同的交互界面**。ACM DIS 2025 论文总结其五大特征：计算共创、扩展设计空间、表达流动性、上下文适应、**合成而非选择**。

**对我们意味着什么**：用户打开「易合同」，每个人看到的首页布局、功能排序、提醒方式都是根据他的行为习惯实时生成的——重收款的看到收款看板，重合同存档的看到快速上传入口，常被催租的房东看到租金日历置顶。**不是用户适应系统，是系统适应每个用户。**

### 发现2：Notion 个人/团队双空间模式 — 完美匹配你的需求

Notion 的架构经过6年验证：**每个用户拥有个人空间（免费/私密）+ 可加入多个团队空间（付费/共享）**。个人空间的合同只有自己可见；切换到企业空间后，自动继承组织架构、角色权限、企业模板。这完全是你说的「个人关联企业，一键切换」的终极形态。

### 发现3：MCP 协议已成为 2026 AI Agent 通用标准

由 Anthropic 提出，2026年已捐赠至 Linux Foundation 治理，获得 OpenAI/Google/Microsoft 全面支持。这意味着用 MCP 协议封装的合同管理工具，可以被任何 AI 客户端（Claude/ChatGPT/Cursor）直接调用——用户可以直接在聊天机器人里说「帮我查一下下个月到期的合同」就能操作。

### 发现4：NocoBase 数据驱动 + 插件微内核 — 插件化的终极形态

22K+ Star，核心思想：**插件不是文件，是数据库里的配置**。每个插件注册自己的数据模型（JSON Schema）、审批流、AI规则，平台自动生成 CRUD API + UI。新增一个行业插件 = 新增一条数据库配置记录，无需改代码。

**数据模型**：
```prisma
model User {
  id            String   @id
  email         String   @unique
  // 个人空间
  personalSpace PersonalSpace?
  // 加入的企业
  memberships   TenantMember[]
}

model PersonalSpace {
  id      String   @id @default(uuid())
  userId  String   @unique
  user    User     @relation(fields: [userId], references: [id])
  // 个人合同直接关联 personalSpaceId
  contracts Contract[]
}

model Tenant {
  id          String   @id
  name        String
  // 企业的行业插件配置
  plugins     Json     // ["landlord", "catering"]
  members     TenantMember[]
}

model TenantMember {
  id       String @id
  tenantId String
  userId   String
  role     String // admin / member / readOnly
}
```

```
   - 识别到金额 > 50万 → 触发高优提醒规则

3. GenUI 根据用户习惯进化
   - 用户反复修改某个字段 → 系统记住偏好
   - 用户常看某个报表 → 下次首页优先展示
   - 用户拒绝某类提醒 → 自动降低该类提醒频率
```

|------|------------------------|------------|
| **界面** | 固定菜单，用户适应系统 | **GenUI 自适应**，系统适应每个用户 |
| **空间** | 只有企业 | **个人空间 + 企业空间**，自由切换 |
| **插件** | 封闭，定制=二次开发 | **数据驱动插件**，新增行业只需配置 |
| **导入** | 手动上传 | **拍照/邮箱/导入** 三通道自动入库 |
| **AI** | 规则引擎 | **MCP多Agent协作**，对话式操作 |
| **个性化** | 管理员统一设置 | **每个人自定义关键词/习惯/提醒** |
| **定价** | 10万-200万，实施重 | ¥0起，按需购买行业插件 |
