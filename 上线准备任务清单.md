## 一、资质与合规（法律硬门槛，必须先办）

### 1.1 ICP 备案（必办，周期 2-4 周）

> SaaS 产品在中国大陆上线必须有 ICP 备案，否则违法。阿里云/腾讯云提供免费代办。

| 任务 | 需要你准备的材料 | 预计耗时 | 操作指引 |
|------|----------------|---------|---------|
| ⬜ **购买域名** | 身份证/营业执照 | 10 分钟 | 阿里云/腾讯云/Namesilo 搜索购买 `yihetong.com` 或类似域名 |
| ⬜ **实名认证** | 身份证（个人）/ 营业执照（企业） | 1-2 天 | 在域名注册商处提交实名资料 |
| ⬜ **服务器购买** | 支付宝/微信支付 | 10 分钟 | 阿里云 ECS 或腾讯云 CVM，最低配 2核4G ¥99/月 |
| ⬜ **提交 ICP 备案** | 域名证书 + 身份证/营业执照 + 网站负责人照片 | 2-4 周 | 阿里云备案中心 → 填写信息 → 幕布拍照 → 提交管局审核 |
| ⬜ **备案成功后悬挂备案号** | — | 30 分钟 | 网站底部添加「京ICP备XXXXXXXX号-X」链接至工信部 |

**推荐路径**：阿里云一站式（买域名 + 买服务器 + 提交备案）

### 1.2 法律文件（上线前必须有）

| 任务 | 说明 | 操作建议 |
|------|------|---------|
| ⬜ **用户协议** | 用户注册时必须同意 | 找一份主流 SaaS（如钉钉/飞书）的用户协议改，重点写清楚：服务内容、费用、免责、争议解决 |
| ⬜ **隐私政策** | 收集用户数据必须告知 | 同上。重点写清楚：收集哪些数据（合同内容/个人信息）、怎么用、怎么删 |
| ⬜ **免责声明** | AI 解析结果不构成法律建议 | 一句话：「AI 解析结果仅供参考，不构成法律意见」|

**推荐做法**：找 ChatGLM/DeepSeek 生成初稿 + 发给你认识的律师朋友审一遍

### 1.3 企业资质（如果你用公司名义运营）

| 任务 | 说明 | 周期 |
|------|------|------|
| ⬜ **营业执照** | 确认公司经营范围包含「软件开发」「互联网技术服务」 | 已有则跳过 |
| ⬜ **对公账户** | 企业版收款需要对公账户 | 已有则跳过 |
| ⬜ **发票资质** | 需要给大企业开发票 | 电子税务局申请 |

## 四、操作建议（怎么做更省事）

### 域名购买建议

```
推荐注册商：阿里云万网（https://wanwang.aliyun.com）
推荐域名：
  yihetong.com      → ¥50-80/年（推荐，.com 最通用）
  yihetong.cn       → ¥30-50/年（.cn 国内注册需实名）
  yihetong.cc       → ¥20-30/年（便宜但不专业）
  
建议策略：先买 yihetong.com + yihetong.cn 两个
  - .com 做官网主域名
  - .cn 做备案域名（国内要求 .cn 备案更顺畅）
```

### ICP 备案省钱省事方案

```
方案一：阿里云一站式（推荐）
  直接在阿里云买域名 + 买服务器 + 提交备案
  最低成本：¥99/年（服务器）+ ¥50/年（域名）= ¥149
  
方案二：Vercel 部署（不需要服务器）
  代码部署到 Vercel（免费）
  但备案需要有国内服务器，所以 Vercel 部署不能免备案
  
结论：无论用哪家托管，只要你面向国内用户，必须 ICP 备案
```

### OpenAI API 省钱技巧

```
1. 注册时用接码平台（sms-activate.org）获取手机验证码
2. 首次充值 $10 起
3. 在控制台设置每月预算上限（防止被盗刷）
4. 备选方案：注册 DeepSeek/豆包 的 API（国内，便宜，无需翻墙）
```

### 如果没有公司

```
个人备案方案：
  1. 用个人身份证进行 ICP 备案
  2. 公众号/小程序的个人版不需要营业执照
  3. 微信支付个人版需要个体工商户执照
  
推荐：如果还没有公司，先以个人名义备案上线
  验证产品可行后再注册公司
```
